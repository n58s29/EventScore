# EventScore 🎯

Outil de pilotage du ROI des écosystèmes numériques territoriaux — SNCF.

## Fonctionnalités

- **Dashboard direction** — KPI, graphiques, classement
- **Scoring avant événement** — Évaluation multicritères, recommandation automatique
- **Suivi résultats** — Saisie post-événement, calcul des coûts unitaires
- **Comparateur** — Tri et benchmark inter-événements

## Lancement local

```bash
npm install
npm run dev
```

## Déploiement GitHub Pages

Le déploiement est automatique à chaque push sur `main` via GitHub Actions.
